# Data in Paper Out


## Current functionality
A simple script implementing the training CNN (Pytorch) and deployment on the WWW (ONNX.js).


## Runtime System Requirements
Windows 10
One or more of the CPU processors:


### Supported CNN
- MobileNet v2
- EfficientNet Lite
